## ğŸ¨ Customization

### Colors
The main color scheme uses AWS branding:
- Primary: `#ff9900` (AWS Orange)
- Dark: `#232f3e` (AWS Dark Blue)
- Light: `#ffffff` (White)

### Adding Assets
Place your images in the `assets` folder and import them in the respective components.

### Modifying Content
Each component has its own file in the `src/components` directory. Update the content directly in these files:
- Update speaker information in `Speakers.jsx`
- Modify schedule in `Schedule.jsx`
- Change registration details in `Registration.jsx`
- Update organizer information in `Organizers.jsx`
